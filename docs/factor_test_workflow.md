# 多因子测试流程文档

## 1. 概述

本文档描述了一套标准化的量化多因子测试流程，适用于遗传规划(GP)挖掘的Alpha因子评估

## 2. 测试流程架构

```
原始因子表达式 (191+101)
        │
        ▼
  ┌─ 因子计算 (Factor Computation) ─┐
  │  解析GP树 → 面板数据 (股票×日期)  │
  └────────────────────────────────┘
        │
        ▼
  ┌─ 数据预处理 (Preprocessing) ────┐
  │  · 去极值 (Winsorize/MAD)       │
  │  · 缺失值填充                   │
  │  · 标准化 (Z-Score)             │
  │  · 行业/市值中性化 (可选)        │
  └────────────────────────────────┘
        │
        ▼
  ┌─ 单因子测试 ────────────────────┐
  │  1. IC分析 (Rank IC / Pearson IC)│
  │  2. 分层回测 (分层收益)          │
  │  3. 多空组合 (Long-Short)        │
  │  4. 因子分布与覆盖度              │
  └────────────────────────────────┘
        │
        ▼
  ┌─ 因子筛选 (Factor Selection) ───┐
  │  · IC_IR 阈值过滤                │
  │  · 相关性去重 (≤0.7)             │
  │  · 换手率控制                    │
  └────────────────────────────────┘
        │
        ▼
  ┌─ 多因子合成 ────────────────────┐
  │  · 等权合成                      │
  │  · IC_IR加权                     │
  │  · 滚动回归合成                  │
  └────────────────────────────────┘
        │
        ▼
  ┌─ 综合评估与可视化 ──────────────┐
  │  · 因子表现热力图                │
  │  · 多空净值曲线                  │
  │  · 分层收益柱状图                │
  │  · IC衰减曲线                    │
  │  · 因子相关性矩阵                │
  └────────────────────────────────┘
```

## 3. 单因子评估指标

### 3.1 IC 分析 (Information Coefficient)

| 指标 | 公式 | 含义 |
|------|------|------|
| **Rank IC** | `corr(rank(factor_t), rank(ret_{t+1}))` | 横截面秩相关系数 |
| **IC Mean** | `mean(IC_series)` | 平均预测能力 |
| **IC Std** | `std(IC_series)` | 预测稳定性 |
| **ICIR** | `IC_mean / IC_std` | 信息系数信息比 |
| **IC > 0 占比** | `sum(IC > 0) / N` | 正向预测概率 |
| **IC 月度均值** | 按月聚合 IC | 月度预测能力变化 |

### 3.2 分层回测 (Stratified Backtest)

- 每日将股票按因子值分为 5/10 组
- 等权/市值加权计算每组次日收益
- 累计净值曲线 → Top-Bottom 多空收益

| 指标 | 含义 |
|------|------|
| **Top 组年化收益** | 最优组表现 |
| **Bottom 组年化收益** | 最差组表现 |
| **多空年化收益** | Top - Bottom |
| **多空夏普比** | 风险调整后收益 |
| **多空最大回撤** | 风险控制指标 |
| **分组单调性** | 收益是否随分组单调递增 |

### 3.3 因子分布分析

- **覆盖度**: 有效因子值占比随时间变化
- **分布形态**: 偏度、峰度
- **自相关性**: 因子自身的时序稳定性 → 衍生换手率

## 4. 多因子合成方法

### 4.1 等权合成
```
composite = mean(factor_1_normalized, factor_2_normalized, ...)
```

### 4.2 ICIR 加权
```
weight_i = max(0, ICIR_i) / sum(max(0, ICIR_j))
composite = sum(weight_i * factor_i)
```

### 4.3 滚动回归合成
- 用过去 N 期的因子值对收益做滚动多元回归
- 每期动态更新权重

## 5. 输出图表（论文级别）

1. **因子 IC 热力图** (`plot_ic_heatmap`)
   - 行=因子, 列=月份, 颜色=IC值

2. **分层累计收益曲线** (`plot_stratified_returns`)
   - 5条曲线 (Q1-Q5) + 多空曲线

3. **IC 衰减曲线** (`plot_ic_decay`)
   - X轴=未来N期, Y轴=IC均值±标准差

4. **因子相关性矩阵** (`plot_correlation_matrix`)
   - 下三角热力图

5. **多空净值与回撤** (`plot_longshort_drawdown`)
   - 双Y轴：净值曲线 + 回撤面积图

6. **因子合成前后对比** (`plot_composite_comparison`)
   - 合成因子 vs 单因子 IR 对比
